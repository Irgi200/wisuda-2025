Web Arsip Project

By: Ikmal Hafiz Az-zaka.

Deskripsi Singkat:

Web yang dibuat khusus untuk media pengarsipan Project para siswa SMP SIBS. Ide awalnya muncul saat adanya kebingungan diantara siswa ketika mengerjakan Project Based Learning(PjBL) yang merupakan program unggulan sekolah ini. Pertanyaan yang kerap muncul adalah "Disimpan dimanakah project-project ini?". Maka, dilakukanlah perenungan panjang oleh pihak siswa. Akhirnya, Dikembangkanlah situs ini sebagai solusi dari permasalahan tersebut.
